#MOTIVATION
- This project is aimed at creating a simple zoom clone as a way to learn more about web development and video streaming.
- The project will be built using Node.js, Express, WebRTC, and Socket.io.
- The project will be built in 3 phases: 
    1. Setting up the server and creating a basic video chat application.
    2. Adding the ability to create and join rooms.
    3. Adding the ability to share the room link and chat with other users.

#PHASE 1
- Initialize a new Node.js project.
- Install Express, Socket.io, and Nodemon.
- Create a basic Express server.
- Create a basic HTML file with video elements.
